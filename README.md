# cisc-124-projects

CISC 124 is a first-year computing course at Queen's University taken after CISC 121. All projects were written in Java.

Assignment 1: Created a grade calculator which generated student's marks based on provided assignment and exam grades, as well as displaying information about the marks in the course.

Assignment 2: Program read data from a file on non-existant songs, categorized them, and as generating a list of 'error' songs which had incorrect provided information.

Assignment 3: Created a 'smart dispatcher' which dispatched Bus, Taxi, and Truck objects to perform their given tasks efficiently.
